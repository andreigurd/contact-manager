# Contact Manager

A simple command-line contact book application.

## Features
- Add contacts with name, phone, and email
- View all contacts in a formatted table
- Search contacts by name
- Delete contacts
- Automatic save/load

## How to Run
```
bash
python contact_manager.py
```
